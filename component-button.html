Buttons

Overview

Buttons are used to trigger actions throughout the app. There are four types: Primary, Secondary, Accent, and Icon. All styles use CSS variables from tokens.css for consistency.

Examples

Code Snippets

<!-- Primary --> <button class="btn btn-primary">Start</button> <!-- Secondary --> <button class="btn btn-secondary">Start</button> <!-- Accent --> <button class="btn btn-accent">Start</button> <!-- Icon --> <button class="btn btn-icon">Add +</button>
